Логические элементы компьютера

В компьютерах все вычисления выполняются с помощью логических элементов — электронных схем, выполняющих логические операции.

Триггером называют электронную схему, которая может находиться только в двух состояниях и способна почти мгновенно переходить из одного состояния в другое. Триггер использует так называемые обратные связи — сигналы с выходов схем "ИЛИ-НЕ" поступают на вход соседней схемы. Для хранения многоразрядных данных триггеры объединяются в единый блок, который называется регистром.

Сумматор предназначен для сложения двоичных чисел. Для сложения многоразрядных чисел сумматоры объединяют в цепочку.
