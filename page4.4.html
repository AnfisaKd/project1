Хранение в памяти вещественных чисел

Существуют два приблизительно равноценных способа представления чисел с плавающей запятой:

1. оптимальный с теоретической точки зрения, в котором целая часть нулевая, а первая цифра дробной части ненулевая

2. более удобный с практической точки зрения, в котором целая часть состоит из единственной ненулевой цифры

В компьютере используется такое представление вещественных чисел с плавающей запятой, при котором значащая часть Z удовлетворяет условию 1<= Z < B, где B — основание системы счисления. Такое представление называется нормализованным.

При кодировании вещественного числа с плавающей запятой хранятся две величины: его значащая часть и порядок.

Значащая часть вещественных чисел хранится в прямом коде.

Для кодирования порядка используется кодирование со смещением: чтобы вообще избавиться от знака порядка, к нему добавляют некоторое положительное смещение d: P = P + d. Величина смещения подбирается так, чтобы число P было всегда положительным.
