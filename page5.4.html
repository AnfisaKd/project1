Процессор

Процессор — это устройство, предназначенное для автоматического считывания команд программы, их расшифровки и выполнения.

Микропроцессор — процессор, изготовленный в виде большой мли сверхбольшой интегральной схемы.

Кроме регистров АЛУ и УУ, в микропроцессоре есть много других регистров.

Регистры общего назначения предназначены для использования программным обеспечением.

Генератор тактовых импульсов служит для организации выполнения команд в компьютере. Он "запускает" очередной такт машинной команды.

Тактовая частота — количество тактовых импульсов в одну секунду.

Разрядность — это максимальное количество двоичных разрядов, которые процессор способен обраюотать за одну команду.

Разрядность шины данных — максимальное количество битов, которое может быть считаноза одно обращение к памяти.

Разрядность шины адреса — количество адресных линий.
